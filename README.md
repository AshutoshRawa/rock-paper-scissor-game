## 📌 About The Project

This is a fun and interactive **Rock Paper Scissors** game built using **HTML, CSS, and JavaScript**.  
The player competes against the computer, and the game updates the result and scoreboard dynamically after every move.

---

## ✨ Features

- ✅ User vs Computer Gameplay
- ✅ Random Computer Choice Generation  
- ✅ Live Score Tracking  
- ✅ Win / Lose / Draw Result Display  
- ✅ Smooth Hover & UI Effects  
- ✅ Simple and Clean Interface  

---

## 🛠️ Built With

- **HTML5**
- **CSS3**
- **JavaScript (ES6)**

## 🎯 How To Play

1. Clone this repository:

```bash
git clone https://github.com/your-username/rock-paper-scissors-game.git
```

2. Open the project folder:

```bash
cd rock-paper-scissors-game
```

3. Run the game by opening `index.html` in your browser.

## 🧠 Game Rules

| User Choice | Computer Choice | Result |
|-----------|----------------|--------|
| Rock      | Scissors       | ✅ Win |
| Rock      | Paper          | ❌ Lose |
| Paper     | Rock           | ✅ Win |
| Paper     | Scissors       | ❌ Lose |
| Scissors  | Paper          | ✅ Win |
| Scissors  | Rock           | ❌ Lose |
| Same      | Same           | 🤝 Draw |
